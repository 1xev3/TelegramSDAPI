# TelegramSDAPI

## Description:
This is telegram bot that using external API:\
https://github.com/AUTOMATIC1111/stable-diffusion-webui/wiki/API

## Setting up (linux)
1. Creating venv:
```bash
python -m venv venv
```
2. Activate venv:
```bash
source venv/bin/activate
```
3. Install dependencies:
```bash
pip install -r requirements.txt
```
4. Create .env file in current folder. Available fields can be finded in config.py
5. Run service:
```bash
python3 app.py
```
